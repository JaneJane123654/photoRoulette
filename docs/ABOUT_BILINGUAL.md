# About / 关于

[English](#english) | [中文](#中文)

## English

Photo Roulette helps you browse and clean your gallery faster using gesture-first interactions.

- Built for quick daily photo cleanup.
- Designed for one-hand use.
- Gives users control over delete safety and silent delete setup.
- Supports multiple languages and in-app language switching.
- Includes optional update reminders based on GitHub Releases.

## 中文

照片轮盘通过“手势优先”的交互方式，帮助你更快浏览和整理相册。

- 适合日常快速筛图与清理。
- 面向单手操作场景设计。
- 提供删除保护与静默删除授权机制。
- 支持多语言与应用内语言切换。
- 支持基于 GitHub Release 的可选更新提醒。
